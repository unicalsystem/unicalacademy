A WordPress LMS Plugin to create WordPress Learning Management System. Turn your WordPress to LMS WordPress Website with Courses, Lessons, Quizzes & more.

== Description ==

Live Course add-on for LearnPress allows admin and instructor to create courses and live video meetings.
The purpose of this add-on is to increase the engagement between instructors and students and make online education experience more interesting.
Zoom and Google Meet can be used to create course and video meeting for your own.
This add-on has simple and friendly interface, we also provide a detailed documentation which makes you feel no difficulty in installing and using Live Course add-on for LearnPress.
